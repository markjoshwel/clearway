# Oversight Clearway Bridge for Gmail

Implementation built around the official Gmail REST v1 API, using Google OAuth
2.0 and calling endpoints such as `users.messages.list` (to list mailbox
messages) and `users.history.list` (to incrementally detect mailbox changes
for polling-based sync).
